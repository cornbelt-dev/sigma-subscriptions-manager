Know your assumptions

Sigma Subscriptions is an open source application interacting with smart contracts on the Ergo blockchain.

This is a beta version, done in the context of Ergohack VI, it is recommended to use it for only small transactions.

The application does not log or collect user data.

Transactions are done live on the Ergo blockchain and are irreversible, they can be followed on the explorer.

All code is open source and available for public review.
